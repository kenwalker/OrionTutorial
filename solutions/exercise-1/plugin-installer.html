Plugin Installer : Shortcut to install Plugins for Orion

An Orion Plugin to easily install a plugin from the Navigator

This is a plugin for Eclipse Orion. This plugin installs a navigator action to install the selected resource as a plugin. It is helpful when developing plugins to be able to try out your raw plugin in Orion locally.

Install into OrionHub or localhost:8080 Orion or another Orion instance
